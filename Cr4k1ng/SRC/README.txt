Cracking GUI demo (educativo)
Archivos:
- cracking_gui.py : aplicación GUI con Tkinter (archivo único)
- wordlist.txt    : wordlist de ejemplo (incluye 's3c')
Instrucciones:
1) Descomprime y abre una terminal en la carpeta.
2) Ejecuta: python cracking_gui.py
3) Usa 'Crear wordlist de ejemplo' o 'Examinar' para seleccionar un wordlist.
4) Ajusta Alfabeto y Max Len para la demo de fuerza bruta.
AVISO: Usar solo con contraseñas/hashes de tu propiedad (educativo).
